# 方剂闪背 — 开发流程

## 开发阶段

| 阶段 | 内容 | 状态 |
|------|------|------|
| Phase 0 | 项目脚手架（Vite + Vue + Router + Pinia） | ✅ 完成 |
| Phase 1 | 数据层（100首方剂数据 + Store + localStorage） | ✅ 完成 |
| Phase 2 | 着陆页（Hero + 功能卡片 + CTA） | ✅ 完成 |
| Phase 3 | 学习卡片页（翻转动画 + 评分按钮 + 快捷键） | ✅ 完成 |
| Phase 4 | 错题本页面 | ✅ 完成 |
| Phase 5 | 收尾（动画 + 响应式 + Favicon） | ✅ 完成 |
| Phase 6 | 项目文档（docs + devlogs + CLAUDE.md） | ✅ 完成 |

## 每阶段验证标准

### 通用标准
- `npm run dev` 能正常启动
- `npx vite build` 构建零错误
- 页面路由切换正常

### Phase 3 验证（核心）
- [x] 卡片点击翻转（3D 动画流畅）
- [x] 空格键翻转
- [x] 1/2/3 键评分
- [x] 左右箭头切换
- [x] 评分后自动跳转下一张
- [x] 进度条实时更新
- [x] 全部学完显示统计
- [x] 刷新页面进度持久化

### Phase 4 验证
- [x] 标记为"模糊/不会"的方剂出现在错题本
- [x] 按错误频率排序
- [x] 单首重新练习
- [x] 清空错题本（有确认弹窗）
- [x] 空状态显示

## 提交规范

```
<type>: <描述>

类型:
- feat:     新功能
- fix:      修复
- style:    样式调整
- refactor: 重构
- docs:     文档
- chore:    构建/工具
```
