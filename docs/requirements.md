# 方剂闪背 — 需求规格

## 产品定位

帮助中医学生高效记忆方剂的 Web 应用，核心是「翻转卡片 + 间隔重复」的学习模式。

## 目标用户

- 中医院校在校生
- 备考方剂学考试
- 需要快速记忆方剂组成、功效、主治、口诀

## 核心功能

### 1. 翻转卡片学习

| 字段 | 说明 |
|------|------|
| 正面 | 方名 + 分类标签 |
| 背面 | 组成、功效、主治、口诀 |

- 点击卡片翻转（3D 动画）
- 键盘快捷键：空格翻卡，1/2/3 对应"不会/模糊/会了"
- 左右箭头切换上/下一张

### 2. 三级评分

| 评分 | 含义 | 颜色 |
|------|------|------|
| 不会 | 完全不记得 | 赭红 #B8372D |
| 模糊 | 有印象但不确切 | 琥珀 #C68B2C |
| 会了 | 牢固掌握 | 松绿 #4A7C59 |

评分后自动记录进度并跳转下一张。

### 3. 学习进度统计

- 顶部进度条：已学 X / 100
- 每轮结束显示统计：会/模糊/不会数量
- 进度持久化到 localStorage

### 4. 错题追踪

- 自动记录"模糊"和"不会"的方剂
- 按错误频率排序
- 支持单首重新练习、全部重新练习、清空错题本
- 空状态友好提示

### 5. 方剂数据

- 100 首常用方剂
- 涵盖 18 个分类
- 数据来源：方剂学（第10版）教材

## 页面清单

| 页面 | 路由 | 说明 |
|------|------|------|
| 着陆页 | `/` | 产品介绍、功能亮点、CTA |
| 学习页 | `/learn` | 翻转卡片学习 |
| 错题本 | `/errors` | 错题列表与复习 |

## 非功能性需求

- 纯前端，无需后端
- 数据持久化到浏览器 localStorage
- 响应式设计，支持手机浏览器访问
- 中文界面
